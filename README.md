# Network-and-web-app-scanning-with-Nessus-and-Nmap
Performed network scans to identify open ports, exposed services, and potential misconfigurations.

Conducted web‑application scanning to uncover outdated software, weak configurations, and common vulnerabilities.

Compared results from both tools to validate findings and understand differences in detection depth.

Documented key vulnerabilities, severity levels, and recommended mitigations.
